Important: Serve from Web Server

This demo must be served from a web server (not opened directly as file://). To run:

  1. Open terminal in this directory
  2. Run: python3 -m http.server 8000 (or npx serve)
  3. Open http://localhost:8000 in your browser

If requests work in curl/Postman but not in browser, this is likely a CORS issue that requires serving from a web server.
